# Cold Reader

A cold, fresh fantasy-style desktop PDF reader built with Tauri 2, React, TypeScript, and PDF.js.

## Requirements

- Node.js 18+
- Rust stable
- Tauri system prerequisites for your OS

## Run in development

```bash
npm install
npm run tauri dev
```

## Build desktop app

```bash
npm run tauri build
```

## Features

- Native PDF file picker
- PDF.js canvas rendering
- Page navigation
- Zoom controls
- Frost/fantasy UI
- Custom Cold Reader icon included at `src-tauri/icons/icon.png`
